# Portal de Jogos 2

Site estático de exemplo sobre jogos.

Como abrir:

- Abrir `index.html` diretamente no navegador; ou
- Rodar um servidor HTTP simples (recomendado):

```powershell
cd "C:\Users\raney\OneDrive\Desktop\python"
python -m http.server 8000
# Abra http://localhost:8000/index.html
```

Arquivos:

- `index.html` — página principal
- `styles.css` — estilos
- `script.js` — interatividade (pesquisa + modo escuro)
